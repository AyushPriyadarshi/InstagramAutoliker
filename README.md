#                                                 INSTAGRAM AUTO LIKER

## REQUIREMENT : 
<ul> 
<li> Mozilla Firefox
<li> Geckodriver 
<li> Selenium
<li> Python 3


~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
                                         DESCRIPTION
           Logs into your account and goto a specific hashtag and likes almost 45 posts in it
           helps to make your Insta active and hence bring back to you, followers
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
